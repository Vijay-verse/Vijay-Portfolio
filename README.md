# PORTFOLIO | Vijay Andhale

A premium, minimal, and high-performance personal portfolio built with a focus on clean aesthetics and modern web standards.

## 🚀 Overview

This project is a complete redesign of my professional digital showcase. It features a sophisticated "slightly dark" theme, editorial typography, and a streamlined user experience designed to highlight my technical skills and creative projects.

## ✨ Key Features

- **Technical Arsenal**: A grid-based showcase of my core competencies, including React, Tailwind CSS, and Git/GitHub.
- **Selected Works**: An immersive gallery of my latest projects with interactive overlays and high-quality mockups.
- **Modern Design System**: Built with a custom vanilla CSS design system, avoiding heavy frameworks to ensure maximum performance and accessibility.
- **Responsive & Accessible**: Fully optimized for all screen sizes, from mobile devices to large desktop monitors.
- **Clean Codebase**: Organized and well-documented HTML, CSS, and JavaScript.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom variables & Grid/Flexbox)
- **Logic**: Modern JavaScript (Intersection Observer API for scroll animations)
- **Icons**: Font Awesome

## 📁 Project Structure

```text
/assets       - Static images and project mockups
/css          - Custom design system and components
/js           - Core application logic and interactions
index.html    - Main entry point
```
